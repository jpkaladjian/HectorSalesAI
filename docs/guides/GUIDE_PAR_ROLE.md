# Guide SDR

## Votre mission

En tant que **SDR**, vous êtes responsable de :
- **Prospecter à distance** (téléphone, email, LinkedIn)
- **Qualifier les opportunités** par visioconférence (R1 à R4)
- **Signer en autonomie** ou **transférer aux BD** pour signatures terrain
- **Alimenter le pipeline** avec des opportunités qualifiées

**Canal de prospection** : Visio uniquement  
**Cycle de vente** : R1 → R2 → R3 → R4 (jusqu'à 4 RDV)  
**Durées autorisées** : 36, 48 ou 60 mois uniquement

---

## Workflow SDR

### Étape 1 : Prospection et création prospect

**Objectif** : Identifier des entreprises cibles et créer les fiches prospects

**Actions** :
1. Recherchez des entreprises cibles (annuaires, LinkedIn, salons)
2. Créez le prospect dans Hector :
   - Menu "Prospects" > "Nouveau Prospect"
   - Renseignez : Nom, contact, email, téléphone
   - Notez la source de prospection
3. Analysez une carte de visite si vous en avez une (extraction auto)

**Best practices** :
- Ciblez les décideurs : DG, DAF, Responsable sécurité
- Vérifiez que l'email est professionnel
- Notez des informations utiles (secteur, taille, enjeux)

---

### Étape 2 : Prise de contact et qualification

**Objectif** : Obtenir un 1er RDV visio (R1)

**Script de prise de contact** :
```
"Bonjour [Prénom],
Je suis [Votre nom] d'ADS GROUP SECURITY. Nous accompagnons 
des entreprises comme [Nom entreprise] dans la sécurisation 
de leurs sites avec des solutions connectées.

J'aimerais échanger 15 minutes avec vous sur vos enjeux 
de sécurité. Seriez-vous disponible cette semaine pour 
une visio ?"
```

**Qualification BANT** :
- **Budget** : L'entreprise a-t-elle un budget sécurité ?
- **Authority** : Parlez-vous au bon décideur ?
- **Need** : Quel est le besoin (vol, vandalisme, conformité) ?
- **Timing** : Projet à court terme ou moyen terme ?

**Si qualifié** :
- Créez l'opportunité dans Hector
- Statut : **R1 Visio Planifié**
- Planifiez le RDV dans votre agenda
- Créez une action de rappel

---

### Étape 3 : Cycle de vente visio (R1 → R4)

#### R1 Visio Planifié → R1 Visio Fait

**Objectif** : Comprendre les besoins et présenter ADS GROUP

**Préparation R1** :
- Consultez le site web de l'entreprise
- Identifiez les enjeux de sécurité du secteur
- Préparez 3 questions ouvertes

**Structure R1 (15-20 min)** :
1. **Intro** (2 min) : Présentation ADS GROUP
2. **Découverte** (8 min) : Besoins, contraintes, budget
3. **Solution** (5 min) : Présentation prestation adaptée
4. **Next steps** (3 min) : Proposition R2 pour approfondir

**Après R1** :
- Si intéressé : Changez statut → **R1 Visio Fait**
- Si non intéressé : Marquez "Perdu"
- Envoyez un email de synthèse sous 24h

---

#### R2 Visio Planifié → R2 Visio Fait

**Objectif** : Présentation détaillée de la solution

**Préparation R2** :
- Préparez une **proposition commerciale chiffrée**
- Sélectionnez la prestation adaptée
- Calculez l'abonnement selon le nombre de sites

**Structure R2 (30 min)** :
1. **Rappel besoins** (5 min)
2. **Présentation solution** (15 min) : Fonctionnalités, bénéfices
3. **Tarification** (5 min) : Abonnement mensuel HT, durée
4. **Traitement objections** (5 min)
5. **Next steps** : R3 pour validation technique ou signature

**Après R2** :
- Changez statut → **R2 Visio Fait**
- Envoyez la proposition par email

---

#### R3 Visio Planifié → R3 Visio Fait

**Objectif** : Validation technique et levée des derniers freins

**Préparation R3** :
- Anticipez les questions techniques
- Préparez une **démo** si possible
- Listez les bénéfices ROI

**Structure R3 (30 min)** :
1. **Questions techniques** (10 min)
2. **Démo solution** (10 min)
3. **ROI et bénéfices** (5 min)
4. **Préparation signature** (5 min) : Documents, processus

**Après R3** :
- Changez statut → **R3 Visio Fait**
- Préparez les documents de signature

---

#### R4 Visio Planifié → R4 Visio Fait → Signé

**Objectif** : Signature du contrat

**Préparation R4** :
- Contrat prêt à signer (DocuSign ou similaire)
- Récapitulatif : Prestation, abonnement, durée
- Conditions générales

**Structure R4 (20 min)** :
1. **Récapitulatif projet** (5 min)
2. **Signature électronique** (10 min)
3. **Prochaines étapes** (5 min) : Déploiement, planning

**Signature réussie** :
1. Changez statut → **Signé**
2. Renseignez la date de signature
3. Hector calcule vos **commissions automatiquement**
4. Félicitations, vous avez signé en autonomie !

---

### Étape 4 : Alternative - Transférer à un BD

**Quand transférer ?**
- Prospect qualifié mais **préfère un RDV terrain**
- Signature importante nécessitant **expertise BD**
- Secteur géographique d'un BD spécifique

**Procédure transfert** :
1. Ouvrez l'opportunité
2. Cliquez "Transférer à un BD/IC"
3. Sélectionnez le **repreneur** (BD disponible)
4. Raison du transfert : "Prospect qualifié nécessitant RDV terrain"
5. Ajoutez des **notes de contexte** :
   - Résumé des échanges R1/R2
   - Profil décideur (DISC si possible)
   - Points de vigilance
6. Cliquez "Demander le transfert"

**Après transfert** :
- Le BD reçoit une notification
- Vous suivez l'état dans "Mes demandes de transfert"
- Si le BD accepte : L'opportunité passe dans son pipeline
- Si le BD signe : Vous touchez une **commission de créateur**

**Best practices transferts** :
- Transférez des opportunités **vraiment qualifiées**
- Donnez un **contexte détaillé** au BD
- Restez disponible pour questions du BD

# Guide BD

## Votre mission

En tant que **Business Developer**, vous êtes responsable de :
- **Développer le terrain** avec des RDVs en face-à-face
- **Reprendre des opportunités SDR** qualifiées
- **Signer rapidement** (R1 ou R2 maximum)
- **Gérer les affaires chaudes** nécessitant un R2

**Canal de prospection** : Terrain uniquement  
**Cycle de vente** : R1 → R2 (2 RDVs maximum)  
**Durées autorisées** : 36, 48 ou 60 mois uniquement

---

## Workflow BD

### Étape 1 : Créer une opportunité terrain

**Cas 1 : Prospection autonome**

Si vous créez une opportunité directement (pas de transfert SDR) :

1. Menu "Pipeline Commercial" > "Nouvelle Opportunité"
2. Sélectionnez le prospect (ou créez-le)
3. **Important** : Statut initial → **R1 Planifié** (terrain)
4. Renseignez abonnement et durée (36/48/60 mois)
5. Planifiez le RDV terrain

**Cas 2 : Reprise transfert SDR**

Si un SDR vous transfère une opportunité :

1. Menu "Transferts" > Onglet "À traiter"
2. Consultez les demandes de transfert
3. Lisez les notes du SDR (contexte, besoins, profil)
4. **Acceptez** le transfert
5. L'opportunité arrive dans votre pipeline en **R1 Planifié**

---

### Étape 2 : Cycle de vente terrain (R1 → R2)

#### R1 Planifié → R1 Fait

**Objectif** : Rencontre terrain pour convaincre et **signer si possible**

**Préparation R1** :
- Lisez les notes du SDR (si transfert)
- Préparez la **proposition commerciale chiffrée**
- Apportez des **supports visuels** (catalogues, tablette avec vidéos)
- Planifiez la démo si pertinent

**Structure R1 terrain (45-60 min)** :
1. **Accueil et présentation** (5 min)
2. **Visite des lieux** (15 min) : Identifiez les points sensibles
3. **Découverte besoins** (10 min) : Questions ouvertes
4. **Présentation solution** (15 min) : Adaptée au site
5. **Tarification** (5 min) : Proposition chiffrée
6. **Closing** (10 min) : Tentative de signature

**Issues possibles R1** :

**Issue A : Signature au R1** ✅
- Changez statut → **Signé**
- Renseignez date de signature
- Commissions calculées automatiquement
- Félicitations, vous avez signé au R1 !

**Issue B : R1 non concluant, besoin R2** ⚠️
- Cloturez le R1 : Bouton "Clôturer R1"
- Indiquez la raison : "Besoin réflexion" / "Validation interne" / "Budget à confirmer"
- L'opportunité devient **Affaire Chaude** automatiquement
- Statut change vers **R1 Perdu - Attente R2**

**Issue C : Perdu définitivement** ❌
- Marquez l'opportunité "Perdu"
- Indiquez la raison (concurrent, budget, timing)

---

#### Gérer les Affaires Chaudes

**Après un R1 non concluant** :

L'opportunité apparaît dans **Menu "Affaires Chaudes"** avec :
- Date limite R2 (calculée automatiquement)
- Jours restants (rouge si < 7 jours)
- Raison non-signature R1
- Actions prévues avant R2

**Vos actions** :

1. **Positionner R2** :
   - Cliquez "Positionner R2"
   - Choisissez la date du 2ème RDV
   - Listez les actions avant R2 : "Envoyer étude de cas", "Appel de relance", "Proposition adaptée"
   - Statut passe à **R2 Planifié**

2. **Relancer** :
   - Cliquez "Relancer"
   - Email automatique envoyé au prospect
   - Personnalisez le message
   - Proposez un créneau R2

**Best practices affaires chaudes** :
- Ne laissez **jamais** une affaire chaude sans R2 positionné
- Relancez dans les **48h** après R1
- Préparez le R2 : nouveaux arguments, offre adaptée, étude de cas

---

#### R2 Planifié → R2 Fait → Signé

**Objectif** : Signature au R2

**Préparation R2** :
- Relisez les notes du R1
- Préparez les réponses aux objections du R1
- Adaptez votre offre si nécessaire
- Apportez de **nouveaux éléments** : étude de cas, témoignage client, ROI calculé

**Structure R2 terrain (30-45 min)** :
1. **Rappel R1** (5 min) : Synthèse des échanges
2. **Nouveaux éléments** (15 min) : Étude de cas, ROI, réponses objections
3. **Proposition finale** (10 min) : Offre ajustée si besoin
4. **Closing** (10 min) : Signature

**Après R2** :
- **Si signature** : Changez statut → **Signé** ✅
- **Si perdu** : Marquez **R2 Perdu** (perdu définitif) ❌

# Guide IC

## Votre mission

En tant qu'**Ingénieur Commercial**, vous êtes responsable de :
- **Signer de nouveaux contrats** (clients importants)
- **Gérer les reconductions** (renouvellements)
- **Accompagner des projets complexes** nécessitant expertise technique
- **Reprendre des opportunités SDR** à fort potentiel

**Canal de prospection** : Terrain + expertise technique  
**Cycle de vente** : R1 → R2 (cycle court)  
**Durées autorisées** : Toutes durées (12 à 60 mois)

---

## Workflow IC

### Différences IC vs BD

| Critère | BD | IC |
|---------|----|----|
| **Cibles** | PME/ETI | Grands comptes |
| **Complexité** | Standard | Complexe, multi-sites |
| **Expertise** | Commerciale | Technique + Commerciale |
| **Durées** | 36/48/60 mois | Toutes durées (12-60) |
| **Cycle** | R1→R2 court | R1→R2 avec validations techniques |

---

### Étape 1 : Nouveaux contrats

**Prospection grands comptes** :
1. Identifiez les comptes stratégiques (>100 salariés, multi-sites)
2. Créez le prospect avec informations détaillées
3. Créez l'opportunité en **R1 Planifié**
4. Préparez une **étude technique** avant R1

**R1 IC** (1-2h) :
- Visite approfondie des sites
- Audit de sécurité
- Identification besoins techniques
- Présentation solution sur-mesure

**R2 IC** (1h) :
- Proposition technique détaillée
- ROI et bénéfices
- Validation contractuelle
- Signature

---

### Étape 2 : Reconductions (renouvellements)

**Objectif** : Renouveler les contrats arrivant à échéance

**Process** :
1. Hector vous alerte 2 mois avant échéance contrat
2. Contactez l'entreprise pour RDV reconduction
3. Proposez :
   - Renouvellement à l'identique
   - Upgrade prestation (nouvelles fonctionnalités)
   - Extension multi-sites
4. Créez l'opportunité de type "Reconduction"
5. Signez le renouvellement

**Best practices reconductions** :
- Contactez **2 mois avant échéance**
- Proposez toujours un **upgrade** (plus de valeur)
- Mettez en avant la **fidélité** (tarif préférentiel)

---

### Étape 3 : Reprises SDR

**Transferts SDR vers IC** :

Les SDR peuvent vous transférer des opportunités si :
- **Projet complexe** (multi-sites, intégration spécifique)
- **Contrat important** (>10 000€ MRR)
- **Expertise technique** requise

**Process identique aux BD** :
1. Menu "Transferts" > "À traiter"
2. Consultez la demande
3. Acceptez si vous avez la capacité
4. L'opportunité arrive en **R1 Planifié**

---

# Guide Chef des Ventes

## Votre mission

En tant que **Chef des Ventes**, vous êtes responsable de :
- **Manager l'équipe BD** (coaching, suivi, motivation)
- **Piloter la performance collective**
- **Contribuer personnellement** aux signatures
- **Optimiser les process commerciaux**

**Rôle double** : Management + Contribution perso  
**Durées autorisées** : 36, 48 ou 60 mois uniquement

---

## Workflow Chef des Ventes

### 1. Management de l'équipe BD

**Suivi quotidien** :
- Consultez le **Dashboard Chef** : Menu "Mes Statistiques"
- Visualisez la performance équipe :
  - Signatures totales équipe
  - MRR généré équipe
  - Affaires chaudes équipe
  - Nombre de BD actifs

**Suivi individuel** :
- Consultez les stats de chaque BD
- Identifiez les BD en difficulté (taux signature < 50%)
- Planifiez des **debriefs hebdomadaires**

**Actions management** :
1. **Coaching BD** : Accompagnez les BD sur RDVs complexes
2. **Formation** : Utilisez Hector IA pour former votre équipe
3. **Motivation** : Célébrez les succès, challengez les objectifs

---

### 2. Contribution personnelle

**Vous vendez aussi !**

En tant que Chef, vous gérez vos propres opportunités :
- Pipeline Terrain (R1 → R2)
- Transferts SDR si besoin
- Affaires chaudes personnelles

**Best practices** :
- Ciblez les **comptes stratégiques**
- Montrez l'exemple (taux signature élevé)
- Intervenez sur **dossiers complexes** (aide aux BD)

---

### 3. Optimisation des process

**Analysez les performances** :
- Identifiez les **bottlenecks** : Où perdez-vous des opportunités ?
- **Taux signature R1** : Objectif 60%+
- **Délai R1→R2** : Objectif < 15 jours
- **Affaires chaudes** : Ne laissez rien traîner

**Améliorations** :
- Formez les BD sur les objections fréquentes
- Créez des **playbooks** pour situations types
- Partagez les **best practices** en équipe

---

### 4. Réunions d'équipe

**Hebdomadaire** (30 min) :
- Revue des signatures semaine
- Partage des succès
- Identification des blocages

**Mensuelle** (1h) :
- Bilan du mois
- Analyse des KPIs équipe
- Objectifs mois suivant
- Formation collective
